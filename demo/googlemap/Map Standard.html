Where should Google Map be used?

1. Map should be shown after users input their address in profile.

2.Maps should be shown in the event page according to the address information from database.

How to achieve the Google Map?

1. For profile:

Users input address and click on 'show map'

or

Show map according to what users have input yet.(prefered)

2.For event page:

Event page will first get information from database and then use function map(address) (function name may be modified later) to show map.
